<h1 align="center">Depth-First Search Practice</h1>

<p align="center">
A simple Python program for practicing depth-first graph traversal.
</p>

<hr/>

<h2>📝 Description</h2>
This project demonstrates a DFS algorithm on a graph using Python.  
The algorithm explores as far as possible along each branch before backtracking, ensuring that every reachable node is visited.  
It records the traversal order and helps visualize how DFS navigates through graph structures.
